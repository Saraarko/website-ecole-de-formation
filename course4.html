Course Overview

You will learn to build, deploy, test, run, and manage cloud-native applications. Covered technologies include cloud fundamentals, GitHub, Node.js, React, CI/CD, containers, Docker, Kubernetes, OpenShift, Istio, databases, NoSQL, Django ORM, Bootstrap, application security, microservices, serverless computing, and more. Upon completion of the program, you will have developed several applications using front-end and back-end technologies and deployed them to a cloud platform using Cloud Native methodologies. You will publish these projects via your GitHub repository to share your portfolio with your peers and potential employer.

what you'll learn

Master the practical skills and the latest tools used by full-stack developers on a daily basis.

Learn to deploy and scale applications using Cloud Native methodologies and tools such as Containers, Kubernetes, Microservices, and Serverless.

Develop software with front-end development languages and tools such as HTML, CSS, JavaScript, React, and Bootstrap .

Build your GitHub portfolio by applying your skills to numerous labs and hands-on projects, including a capstone project.
